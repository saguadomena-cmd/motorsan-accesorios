Configurar Base de datos en la nube

Para compartir datos en tiempo real entre Recambios y Ventas sigue estos pasos. Solo se hace una vez.

1
Crea un proyecto Firebase gratuito

Ve a console.firebase.google.com → "Añadir proyecto" → ponle el nombre que quieras (ej: motorsan-pedidos) → desactiva Google Analytics → Crear.

2
Activa Realtime Database

En el menú izquierdo: "Build" → "Realtime Database" → "Crear base de datos" → elige la región más cercana (europe-west1) → modo "Empezar en modo de prueba" → Activar.

3
Copia la URL de tu base de datos

Aparece arriba en color gris, con el formato: https://TU-PROYECTO-default-rtdb.europe-west1.firebasedatabase.app

4
Copia tu API Key

Haz clic en el icono ⚙️ junto a "Project Overview" → "Configuración del proyecto" → pestaña "General" → sección "Tus apps" → haz clic en "</>" (web) → registra la app → copia el valor de apiKey.

Termina en .firebasedatabase.app — sin barra final
Usar solo en este ordenador (sin nube)
